For the site owner — you!

Auto-Update via GitHub

Host AILoom on GitHub Pages for free. Update by committing the new file.

1
Create GitHub Repo
Go to github.com → New repository → Name it ailoom → Public → Create
2
Upload Your File
Click "Add file" → Upload files → Drag your ailoom.html → Rename to index.html → Commit
3
Enable Pages
Settings → Pages → Source: Deploy from branch → Branch: main → Folder: / (root) → Save
4
Update Anytime
Ask Claude for changes → Download new HTML → Go to repo → Click index.html → Edit → paste new code → Commit → Live in ~30s
Your live URL will be: https://YOUR-USERNAME.github.io/ailoom/

To update after every Claude session: Download the new HTML → In GitHub click index.html → click the ✏️ pencil icon → select all → paste → "Commit changes" → Done! Your site updates in about 30 seconds.

Pro tip: Enable Firebase Auth in the Firebase Console and add your GitHub Pages domain (e.g. yourname.github.io) to "Authorised domains" so Google Sign-In works on your live site.
